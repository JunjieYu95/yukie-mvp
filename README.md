# Yukie MVP Blueprint (Repo Skeleton + Design Docs)

This repository is a blueprint-only MVP scaffold for your “Yukie” experimental lifestyle system.

Key folders:
- `docs/` — PRD, architecture, protocols (YWAIP v1 + YNFP v1), security, UX, deployment, testing, roadmap
- `packages/yukie-core/` — Yukie Core placeholders (router/policy/inbox)
- `packages/assistants/` — domain assistant placeholders (NL → function-call)
- `packages/services/` — capability service placeholders (allow-listed functions behind APIs)
- `packages/shared/` — shared protocol/auth/observability placeholders
- `infra/` — deployment placeholders
- `examples/` — example curl payloads

Start reading: `docs/00_overview.md` then `docs/02_architecture.md`.
